# Release Notes v1.3.17

## Overview
This release consolidates the settings menu, redesigns the POV Names configuration interface to match the spelling checker's user words layout, adds the ability to add words to POV Names directly from the spelling right-click menu, ensures that names in the POV list are automatically ignored by the spellchecker, and updates all offline and online user documentation.

## Simple Change Notes
* **POV Settings Consolidation**: Combined the POV pronoun filter and character names list under a single unified "POV Settings" tab.
* **POV Names List Redesign**: Changed the character names input from a comma-separated string to an interactive Listbox design with scrollbar and Add/Remove buttons.
* **Right-Click Integration**: Added an "Add to POV Names" action to the right-click spellchecking context menu.
* **Spellcheck Ignored Names**: The spelling checker now automatically ignores all character names added to the POV settings list.
* **Documentation Sync**: Updated both the online GitHub wiki manual and the offline local HTML help documents (`file_menu.html`, `spell_checking.html`) to reflect the settings reorganizations.
